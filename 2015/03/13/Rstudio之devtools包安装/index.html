##关于安装devtools包
在RStudio中安装devtools的时候提示安装错误，可以用如下方法进行解决：
sudo apt-get install libcurl4-openssl-dev

##注意
安装devtools之前，建议先安装RCurl以及httr
install.packages(“RCurl”)
install.packages(“httr”)
install.packages(“devtools”)
